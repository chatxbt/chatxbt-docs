# Setting your default protocols

When an action is given without adding a protocol to which to execute the command. the protocol will use the default settings of that action

<figure>
  <img src="/settings.png" alt="setting" />
  <figcaption>example showing settings</figcaption>
</figure>

For example:
`swap 1 ETH and buy $pepe for me`
\
this command will default to your set AI Action of swap which will interact with 1-inch protocol
